# Code of Conduct

## Our standard

We want SpatialFlow Agent to be a respectful, practical, and welcoming open-source project.

Please participate in ways that:

- assume good intent
- give direct feedback without hostility
- keep technical disagreement focused on the work
- avoid harassment, discrimination, or personal attacks
- protect other people's time, privacy, and safety

## Scope

This applies to:

- issues
- pull requests
- discussions
- release notes
- linked community spaces used for project collaboration

## Reporting

If someone is making the project unsafe or hostile, reach out privately through GitHub contact channels when possible.

When reporting, include:

- what happened
- where it happened
- any relevant links or screenshots

## Maintainer response

Project maintainers may edit, hide, lock, or remove content that breaks these expectations, and may restrict repeat offenders from participating.

The goal is not punishment theater. The goal is to keep collaboration usable for everyone.
